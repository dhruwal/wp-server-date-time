This plugin shows the server local current date time & timezone in the upper right of your admin screen on every page.

== Description ==

This plugin is useful when you want to know what date time & timezone is it on server where wordpress is hosted if you are in other timezone.
Output looks like:

Server Date/Time: Sat 06th Dec,2008 11:17 am
Server Timezone: America/Phoenix, (MST -07:00)

== Installation ==

1. Upload entire `wp-server-date-time` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. There is no options page, just activate and see the output on right most top corner
